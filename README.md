# Simple-Calculator-App
Simple calulator app using HTML, JS, and CSS. 

Files
- `index.html` — main UI. Open this in a browser to use the app.
- `styles.css` — styles.
- `index.js` — application logic
- `dateUtils.js` — shared date/time parsing and validation
